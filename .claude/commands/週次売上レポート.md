先週分の売上CSVを集計して、グラフ付きレポートを作成してください。

## 手順

1. `売上/` フォルダの中にある最新のCSVファイルを読み込む
2. 商品別の合計、日別の推移を集計する
3. 集計結果をもとに、`report.html` を作成する
   - 商品別の合計は棒グラフ
   - 日別の推移は折れ線グラフ
   - タイトルと集計対象期間を先頭に表示する
   - 集計結果から読み取れることを、一言まとめコメントとして自動生成し末尾に添える
4. デザインは `style.css` と同じトンマナ（白背景・ネイビー#1a365d・ブルー#3182ce）に合わせる
